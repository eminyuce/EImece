UI Icons

MStore includes 2 main packs of font icons:

  • Fetaher Icons - for user interface.
  • Flaticon Icons - for social network links.
Adding Feather Icons:
List of Feather Icons 
<i data-feather="iconName"></i>
Adding Social (Flaticon) Icons:
List of Social Icons 
<a href="#" class="social-btn sb-brandName">
  <i class="flaticon-brandName"></i>
</a>
