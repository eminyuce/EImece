Button group

—  Groups a series of buttons together on a single line. See Bootstrap documentation.

Basic example

  • Result
  • HTML
  • Pug

<!-- Solid button group -->
<div class="btn-group" role="group" aria-label="Solid button group">
  <button type="button" class="btn btn-primary">Left</button>
  <button type="button" class="btn btn-primary">Middle</button>
  <button type="button" class="btn btn-primary">Right</button>
</div>

<!-- Outline button group -->
<div class="btn-group" role="group" aria-label="Outline button group">
  <button type="button" class="btn btn-outline-primary">Left</button>
  <button type="button" class="btn btn-outline-primary">Middle</button>
  <button type="button" class="btn btn-outline-primary">Right</button>
</div>

<!-- Pill solid button group -->
<div class="btn-group" role="group" aria-label="Pill solid button group">
  <button type="button" class="btn btn-pill btn-primary">Left</button>
  <button type="button" class="btn btn-pill btn-primary">Middle</button>
  <button type="button" class="btn btn-pill btn-primary">Right</button>
</div>

<!-- Pill outline button group -->
<div class="btn-group" role="group" aria-label="Pill outline button group">
  <button type="button" class="btn btn-pill btn-outline-primary">Left</button>
  <button type="button" class="btn btn-pill btn-outline-primary">Middle</button>
  <button type="button" class="btn btn-pill btn-outline-primary">Right</button>
</div>
